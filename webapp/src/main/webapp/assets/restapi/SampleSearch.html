REST API

Sample Search inside a SampleGroup (beta version)

  • It returns all the samples that are inside the SampleGroup? parameter and that also match the query(e.g.  http://wwwdev.ebi.ac.uk/biosamples/xml/groupsamples/SAMEG23004/query=cancer);
  • It is possible to use paging (  e.g.http://wwwdev.ebi.ac.uk/biosamples/xml/groupsamples/SAMEG23004/query=cancer&page=3&pagesize=10);
  • Always return a result summary information;


Authentication

N/A



Arguments

query (required)
sortby (optional - default value=relevance)
sortorder (optional - default value=descending)
page (optional - default value=10)
pagesize (optional - default value=1)



Example Request

Get xml/groupsamples/SAMEG23004/query=cancer&page=2&pagesize=20
