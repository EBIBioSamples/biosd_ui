REST API

Group Search (beta version)

  • It returns all the groups that match the query (e.g.  http://wwwdev.ebi.ac.uk/biosamples/xml/group/query=cancer);
  • It is possible to use paging and sorting (e.g.  http://wwwdev.ebi.ac.uk/biosamples/xml/group/query=cancer&sortby=relevance&sortorder=descending&page=2&pagesize=50);
  • Always return a result summary information;


Authentication

N/A



Arguments

query (required)
sortby (optional - default value=relevance)
sortorder (optional - default value=descending)
page (optional - default value=50)
pagesize (optional - default value=1)



Example Request

Get /xml/group/query=cancer&page=2
