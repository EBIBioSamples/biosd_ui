Programatic Interface

BioSamples Database provides an interface that is suitable for programmatic queries. This consists of four entry point URLs ( descibed below), each of which returns an XML response.

If you have any problems or suggestions please contact us.

Sample Accession

  • Returns information about a single specified sample.
  • http://www.ebi.ac.uk/biosamples/xml/sample/ACCESSION where ACCESSION is the BioSample Database accession of the sample of interest.
  • e.g. http://www.ebi.ac.uk/biosamples/xml/sample/SAME315173
  • more information

Group Accession

  • Returns information about a specified group.
  • http://www.ebi.ac.uk/biosamples/xml/group/ACCESSION where ACCESSION is the BioSample Database accession of the group of interest.
  • e.g. http://www.ebi.ac.uk/biosamples/xml/group/SAMEG82620
  • more information

Find Groups

  • Returns group accessions that match the specified search query.
  • http://www.ebi.ac.uk/biosamples/xml/group/query=QUERY where QUERY is the term to query for
  • e.g. http://www.ebi.ac.uk/biosamples/xml/group/query=leukemia
  • Results are returned in one or more pages and will always return result summary information.
  • Can use advanced query features.
  • more information

Find Samples Within a Group

  • Returns sample accessions that match the specified search query and are in the specified group.
  • http://www.ebi.ac.uk/biosamples/xml/groupsamples/ACCESSION/query=QUERY where ACCESSION is the group to query within and QUERY is the term to query for.
  • e.g. http://www.ebi.ac.uk/biosamples/xml/groupsamples/SAMEG82620/query=leukemia
  • Results are returned in one or more pages and will always return result summary information.
  • Can use advanced query features.
  • If query is blank, all samples within that group are matched.
  • more information
